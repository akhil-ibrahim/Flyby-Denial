# Flyby-Denial 🧬
[View the Repository on GitHub](https://github.com/luquelab/Flyby-Denial)

## Project Description
Flyby-Denial is a bioinformatics pipeline developed for BIL552 at the University of Miami. It automates the physicochemical profiling and hierarchical clustering of protein sequences to analyze structural relationships.
